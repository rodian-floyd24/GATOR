## How to Run in Snowflake (Snowsight)

- Repository/branch: `main` of `https://github.com/rodian-floyd24/GATOR`.
- Streamlit location: folder `streamlit_app`.
- In Snowsight Streamlit Apps: select repo `GATOR`, fetch branch `main`, choose folder `streamlit_app`, then Run.
- App features: date range filter, state multiselect, row limit slider; each tab runs one of the five Week 11 queries and offers CSV download.
